# yell
あなたにエールを送るAIエージェント

## 必要なパッケージのインストール

```bash
pip install
```

## Google API Keyの設定

- Gemini APIを利用します。

### mac, linux

```bash
export GOOGLE_API_KEY="あなたのAPIキー"
```

### windows(powershell)

```powershell
$env:GOOGLE_API_KEY="あなたのAPIキー"
```

## 使い方

使い方 (How to Run)
パターン1: ファイルを渡して実行
今日やったことやメモが書かれたテキストファイル (todo.txt とか) がある場合。

Bash

python yell.py todo.txt
挙動: ファイルを読み込んで、いきなり「分析→褒める」モードに入ってくれる。「おお、これ全部終わらせたの？すごいじゃん！」みたいに。

パターン2: 何もなしで実行 (インタビューモード)
手ぶらで起動する場合。

Bash

python yell.py
挙動: 「今日もお疲れ様。今日はどんな一日だった？」と優しく聞いてくる。チャットで「エラーが消えなくて辛かった…」みたいに打ち込むと、それを受け止めてから分析してくれる。

さあ、これで君だけの「深夜の相棒」が生まれたよ。 今夜、寝る前に一度起動してみて。きっと、いい夢が見られるはずだ。

## オプション

### デバッグモード

`-d`でデバッグモードになります。Gemini APIとのやりとりをすべて表示します。

```
python yell.py -d 2026-01-20.txt 2026-01-21.txt
```
